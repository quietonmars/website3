After opening the source code with Pycharm, Sublime or any preferred text editor.
Go to the terminal and install requirements.txt using this code
pip install -r requirements.txt

this will install the required packages for the website

After everything is successfully installed, run main.py

when running main.py, system will create a department called "Management" and create a default superadmin account.
